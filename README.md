# 📊 股权激励预算管理 Demo

一个基于 Streamlit 构建的动态股权激励预算管理系统，支持职级标准设置、HC规划、员工管理和股权授予等功能。

## ✨ 功能特性

- **🎯 职级标准管理**：设置不同职级的股权授予标准
- **📋 HC规划管理**：基于部门规划招聘需求
- **👥 员工全生命周期管理**：入职、离职、晋升等操作
- **📈 股权授予追踪**：实时监控股权授予情况
- **💰 股票池动态监控**：可视化展示股票池使用情况
- **⚠️ 预警系统**：自动检测股票池余额不足等风险
